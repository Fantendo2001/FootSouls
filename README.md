# 数天训练营线上组工程仓库

#### 介绍
数天训练营线上组

#### 项目拷贝步骤

1.  安装Git
2.  安装Fork（Git的图形化界面，便于操作）
3.  克隆项目到本地并尝试运行

#### 使用说明

详情见拷贝后的项目工程使用说明。

#### 注意事项
1. 从网上下载的素材时最好在本地运行，觉得好要用的文件后进行上传，因Gitee仓库有容量限制，
传输文件有大小限制，仓库大小约为1-3G，因此不可能存放无用的资产，酌情添加资产。
2. 注意单个文件大小最好小于50m，因其Gitee限制可能无法上传大于50m的文件。
